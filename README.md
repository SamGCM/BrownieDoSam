# Site de venda/exposição
WebApp feito para exposição ou venda de produtos.

Tipos:
A branch MAIN é o site na forma de exposição dos produtos, para influenciar e dar confiança a compra do consumidor da marca ou produto.

A branch Sistema-de-loja como já diz o nome, tem uma for de compra dentro do site. A escolhida foi Link de pagamento da PagSeguro, mas pode ser mudada conforme o gosto do proprietário do site. Além de incluir localização de pontes de venda, usando mapa do GoogleMaps. Importante lembrar que o endereço do comprador, que é usado para entregar o produto, só se consegue pelo site da PagSeguro, assim ajudando a ter uma organização de sequência de compras feitas e entregas.
